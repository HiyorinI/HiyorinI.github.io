# 图片资源目录

请将以下图片资源放置在此目录：

## 必需的图片

- `logo.png` - 网站 Logo (建议尺寸：200x200)
- `logo.png` - 网站 Logo (建议尺寸：200x200)
- `home-background-light.jpg` - 首页背景图 (亮色模式)
- `home-background-dark.jpg` - 首页背景图 (暗色模式)
- `SBR.jpg` - 示例图片
- `shoko.jpg` - 作者头像

## 用户头像

在 `users/` 子目录下放置用户头像图片：
- `user1.jpg` 到 `user6.jpg`

## 展示案例图片

在 `showcase/` 子目录下放置案例图片：
- `project_management.png`
- `data_viz.png`
- `music_player.png`

## 组件截图

在 `components/` 子目录下放置各组件的截图，按组件分类：
- `button/` - 按钮组件截图
- `slider/` - 滑块组件截图
- `combo_box/` - 组合框截图
- 等等...

## 图标

网站使用的图标可以在这里放置，或者使用 CDN 链接。
